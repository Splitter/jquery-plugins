SV ImageTooltip image preview/tooltip plugin

> image preview/tooltip Plugin

As you hover over an element with the mouse the script will show an image preview in a tooltip fashion.

Example:

Usage:

To begin you must include both the jQuery file and the plugin file itself.
(usually done in the head of the document). 


<script type="text/javascript" src="jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="jquery.svImageTooltip.js"></script>


the HTML 

<div rel="./images/5.jpg"></div>

The image preview/tooltips can be attached to almost any html element, you just need to set the 'rel' attribute equal to the path of the image you wish to be displayed when the element is hovered over.



Initializing the script 

var opts = {
	border: '4px solid #555', //Border around image popup
	offsetX: 20, //Horizontal offset
	offsetY: 20 //Vertical offset
};
$('YOUR_SELECTOR').svImageTooltip(opts);


border is if you wish to add a colored border around the image preview/tooltip

The offsetX is the horizontal offset from the current mouse position

The offsetY is the the horizontal offset from the current mouse position
