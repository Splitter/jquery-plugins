SV TitleTooltip simple little title tooltip plugin

> title tooltip Plugin

As you hover over an element with the mouse the script will display the title of the element in a tooltip balloon.

note: the script comes with a light and dark css style(uses no images), both of which can be displayed above or below the element that is recieving the tooltip.

Example:

"Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum

Usage:

To begin you must include both the jQuery file, the plugin file itself and the accompanying stylesheet provided.
(usually done in the head of the document). 


<script type="text/javascript" src="jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="jquery.svTitleTooltip.js"></script>
<link rel="stylesheet" href="./svtooltip.css" />


the HTML 



<span class="tooltip_lighttop" title="This is our tooltip text~!">Some random text</span>

or 

<span class="tooltip_darkttop" title="This is our tooltip text~!">Some random text</span>

or 

<span class="tooltip_lightbottom" title="This is our tooltip text~!">Some random text</span>

or 

<span class="tooltip_darkbottom" title="This is our tooltip text~!">Some random text</span>


Adding the 'tooltip_lighttop' CSS class to an element will present the tooltip with the light theme and it will show above the element, and a class of 'tooltip_darkbottom' will present the tooltip with the dark theme and it will show below. The other two classes follow the same thought process.

This script is different then the others presented with this pack as it gets pre-initialized when it is included, adding the title and appropriate css class to an element is all that needs to be done for the script to be functional after including it.
