SV AnimateHover the universal hover animation plugin

> What is svAnimateHover?

Lets face it the majority of hover animation effects take place by animating a specific(or set of specific) CSS attributes from one position to another set position as the mouse hovers over an element and then animate the same attribute('s) back to the(their) original positions as the mouse leaves the element.

This leaves a developer(thats using jQuery) to either create a similar block of code for each hover animation or use a plugin for each type of hover effect(which consist of similar code bases).

svAnimateHover is a universal hover animation plugin meant to reduce the size of the code used, be it the amount of plugins needed or custom code created, to implement hover effects.

Many examples are displayed lower in the page.

> What svAnimateHover is NOT

svAnimate hover is NOT a plugin to create a specific effect. It is NOT a plug and play effect to add to your website. It IS meant for someone with a working html/css/javascript knowledge set who wants to simplify his code base with a simple versatile plugin.

> Usage

To begin you must include both the jQuery file and the plugin file itself.
(usually done in the head of the document). 


<script type="text/javascript" src="jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="jquery.sv_animateHover.js"></script>

Or with the easing plugin if you wish(optional) 

<script type="text/javascript" src="jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="jquery.easing.1.3.js"></script>
<script type="text/javascript" src="jquery.sv_animateHover.js"></script>


To actually use the script just call svAnimateHover on the objects you wish to effect. The script takes an options object as a parameter and there are 5 options the script takes. 



var options={
	changes:{},//Object containing CSS changes to make
	speed:300, //Speed of the animation
	selector:'',//OPTIONAL selector for child of element being hovered over
	easing:'easeInBack', //OPTIONAL easing function
	onstart:function(){},//OPTIONAL on start callback function
	onend:function(){}//OPTIONAL on end callback function
	};
$('YOUR_SELECTOR').svAnimateHover(optionss);

The changes object will take the position to animate the attribute to. So for example if you wanted to move the 'margin-left' of an element by 15px and it already has a 'margin-left' of 5px set, you would pass 20px(15 you want to add + the 5 it has) to the script.

The speed is the duration of the animation.

The easing is an optional easing function(if you are using the jquery easing plugin).

The selector option is for times you wish to animate a child element when the parent is hovered over(see examples below)

The last two options are optional callback functions. The onstart callback function gets executed as the users mouse hovers over an element, and the onend is executed as the animations finish after the mouse leaves an element. They are called in scope of the objects being hovered over so the 'this' variable within the functions will refer to the actual element being hovered over.

> Drop Down Menu Example

This example makes use of the onstart and onend callback functions.

Example:

This example is based on an unedited css dropdown menu downloaded from cssmenumaker.com

License doesn't allow for the file to be included, but if downloaded from cssmenumaker link above, the code below will work flawlessly with the downloaded menu. And if CSS dropdown is created in a similar fashion the code will be pretty much the same.

The Code:

var opts={
		changes: {'opacity':1},
		selector:'ul',
		speed:	 700,
		onstart:function(){
				$(this).children('ul')
					.css({'display':'block'});
		},
		onend:	function(){
				$(this).children('ul')
					.css({'display':'none'});
		}
	};
$('.menu > ul > li').each(function(){	
	$(this).children('ul').css({ opacity: 0 });
	$(this).svAnimateHover(opts);
});

> Thumbnail Expanding Example

This is a slightly altered version of the effect presented in this tutorial.(Just a pretty cool effect)

Example:

The Code:

var opts={
	changes:{
		'top': '-32px',	'left': '-32px',
		'width': '174px','height': '174px'
	},
	speed:300,
	easing:'easeOutBack',
	onstart:function(){
		$('ul.thumb li').css({'z-index' : '0'})
		$(this).addClass("hover").parent()
					.parent().css({'z-index' : '10'})
	}
};
$('ul.thumb li img').svAnimateHover(opts);
