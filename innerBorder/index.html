SV InnerBorder A simple inner border plugin

> Example and Code

This plugin allows you to easily dynamically add a colored semi-transparent inner border to html elements.

Usage:

To begin you must include both the jQuery file and the plugin file itself.
(usually done in the head of the document). 


<script type="text/javascript" src="jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="jquery.svInnerBorder.js"></script>


Initializing the script 

var options = {
	thickness: 3, //border thickness
	color: '#ff1467', //color of border
	opacity: '0.4', //opacity of border
	replace:false //whether to replace existing innerBorders
};
$('.example3 li img').svInnerBorder(options);


thickness is the thickness in pixels of the border.

color is the CSS(hex) color code for the border color.

opacity is the opacity you want border set to.

replace is whether the script should remove old innerborders before adding new, if set to false it will stack the new border ontop of the old. It defaults to true.
