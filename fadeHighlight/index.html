SV FadeHighlight fade highlighting plugin

> Fade Highlighting Plugin

Hover the mouse over one element and other similar elements fade out to provide a 'highlight' effect. A very useful plugin for galleries as well as portfolio pages.

Example:

Usage:

To begin you must include both the jQuery file and the plugin file itself.
(usually done in the head of the document). 


<script type="text/javascript" src="jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="jquery.svFadeHighlight.js"></script>


Initializing the script 

var options = {
   opacity: 0.4, //Opacity to fade other elements out to
   speed:600, //Speed for fade animation
   selector: '' //OPTIONAL Selector - defaults to select similar elements
};
$('YOUR SELECTOR').svFadeHighlight(options);


opacity is the opacity you want to fade the other elements to.

speed is the duration for the fade animation

selector is an optional paramater. To explain better I will explain how the script functions by default. The example above is initialized like this: 


$('div.div_block').sv_highlight(options);



'div.div_block' is the selector that points to the three blocks above. By default the script will fade out all other elements matched by that selector not including the one currently being hovered over. You can pass in a custom selector and the script will fade out elements that match that selector you pass in instead of the other elements that match the original selector.
